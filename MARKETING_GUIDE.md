# Marketing Guide (B2B SaaS, Utility + Energy)

## 1) Primary Goals
- Convert executives to demo requests.
- Communicate ROI and technical credibility.
- Maintain fast, responsive UX.

## 2) Target Personas

### Utility Executive
- Concerns: reliability, compliance, ROI, timeline, vendor risk.
- Motivation: credible partner and clear business case.

### Program / Grid / DERMS Manager
- Concerns: integration, workflows, reporting, support.
- Motivation: automation and low-friction rollout.

## 3) Messaging Framework
- Lead with business impact.
- Support with technical detail.
- Keep paragraphs short and scannable.

## 4) Page Structure
- **Hero:** Value prop + demo CTA
- **Stats:** Quantified outcomes (only with verified sources)
- **Feature Grid:** Capabilities and integrations
- **Case Study:** Business impact
- **Proof:** Certifications, compliance, partners
- **CTA:** Demo request + contact form

## 5) CTA Library
- Request a demo
- Talk to an expert
- View the platform

## 6) Claims and Compliance
- Use real numbers only with citations.
- No unverified performance claims.
- Maintain proof-first copy style.

## 7) Performance and UX
- Mobile-first layout
- Accessible forms and buttons
- Avoid heavy animations

## 8) Major Activity Coverage
Ensure page content supports these typical activities:

- Executive decision support (ROI framing, risk reduction)
- Program operations (workflow efficiency and reporting)
- Implementation planning (timeline clarity and integration readiness)
- Compliance readiness (auditability and documentation)

## 9) GPT-Style Application Coverage
Include clear descriptions of GPT-style work items without making performance
claims:

- Document drafting and review support
- Summarization and structured outputs
- Triage and routing assistance

## 10) Additional Sections to Consider
- Security and governance overview
- Integration and deployment model
- Evaluation and monitoring summary

## 11) Voice + Visual Agent Positioning
- Memory and knowledge governance (what is stored and why)
- Real-time orchestration across tools and workspaces
- Visual interaction layer for clarity and trust

## 12) Integration Messaging Examples
- "Embedded voice assistance across web and mobile experiences."
- "Workspace-safe actions with audit trails and approvals."
- "Low-latency voice with transparent transcription and controls."
